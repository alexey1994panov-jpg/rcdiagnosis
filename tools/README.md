# Tools

Папка `tools/` содержит вычислительное ядро проекта и тесты.

## Основные подпапки
- `core/` — симуляция, движок детекторов, топология, расчет флагов.
- `variants/` — реализации вариантов ЛЗ/ЛС (фазовые фабрики).
- `exceptions/` — исключения и подавления детекторов.
- `station/` — станционная модель и генерация конфигураций.
- `tests/` — автотесты ядра и регрессии.
- `api_contract/` — контракт API и self-check утилиты.
- `docs/` — аудит, техдолг, handoff-документы.

## Точка входа ядра
- Симуляция: `tools/core/sim_core.py`
- Детекторы: `tools/core/detectors_engine.py`
- Флаги: `tools/core/flags_engine.py`

## Запуск тестов
```powershell
python -m pytest -q tools/tests
```

## Примечание
`legasy/` не является источником правды для текущего движка и используется только как справочный архив.
