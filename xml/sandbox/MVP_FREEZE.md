# MVP Freeze: Mnemoscheme Sandbox

Дата фиксации: 2026-02-16

## Scope (заморожено)

- Отрисовка: RC, switches, signals, linked indicators.
- Редактор сценариев в правой панели (шаги, черновик шага, применение).
- Импорт/экспорт JSON (`events` + опционально `steps`).
- Проигрывание по шагам (`назад`, `вперед`, `в начало`).
- Панорама/масштаб схемы (курсор/рука).

## Acceptance checklist

- [x] Недостижимые ветки стрелочной секции остаются нейтральными.
- [x] Цвет RC соответствует состояниям (`3/4/5/6/7/8`).
- [x] Цвет подписей стрелок соответствует контролю (+/-/потеря).
- [x] Linked indicators отображаются и кликаются.
- [x] Правая панель прокручивается независимо от схемы.
- [x] Добавляемый шаг поддерживает edit/delete по строкам.
- [x] Начальное время шага `t=1`.
- [x] Дублирование предыдущего шага работает.

## Regression set

Использовать сценарии из `xml/sandbox/test_scenarios`:
- `01` ... `10`.

## Non-goals (следующий этап)

- Интеграция sandbox в прод-контур.
- Привязка к реальным детекторам ЛЗ/ЛС в runtime.
- Полная UX-полировка редактора сценариев.

## Change policy

Пока статус freeze активен:
- только bugfix,
- без изменения формата сценариев,
- без изменения базовой цветовой семантики состояний.
